The SwissBall

|



The SwissBall is a wireless spherical input device that functions as a discrete and continuous controller. It was created using FormSense, a DIY fabrication method to embed touch and pressure sensing into interactive prototypes of varying shapes.

Publication

Derrek Chow, Gracie Xia, Jasmine Ou. FormSense: A Fabrication Method to Support Shape Exploration of Interactive Prototypes. The ACM Symposium on User Interface Software and Technology (UIST 2022). 3 pages.
